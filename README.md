## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/yourusername/DigiPro.git
cd DigiPro
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run app.py
```
